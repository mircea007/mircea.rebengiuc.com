BAC+

I made this website in colaboration with my cousin, Tiberiu. In romania news outlets are obsessed with the exam at the end of the last school year: Bacalaureat, commonly called BAC. My cousing thought that he should make a website with exam statistics and give it to the media. He told me about it and I asked if I could work on it too. He did the statistics and I focused more on the web design. When the 2021 edition of the exam came along it ended up being a success. Over 12000 people saw the website and we were featured on 30 different media outlets.

Basic 3D graphics

Cristian Francu (my teacher) in 2021 held a few lessons about some very simple 3D graphics. We used SDL, a very simple graphics library and worked our way up. First we learned how to draw lines eficiently with Brezenham and then gradients, then we learned how to generate 3D shapes and render them. I ended up making this program in C. This program can render seven shapes (you can cicle through them using the arrow keys) and you cand rotate them to see what happens (Q, A, S, D, 0, 9). There is also an option to change the projection function by pressing P

Airborne

Airborne is a multiplayer .io game i made during the 2019 summer vacation. I made it using Node.js and HTML canvas. Each player is a plane, you can shoot over players and your score increases evry time you kill someone. The game features: a chat, smoke particle effects, lag and bugs.
